16 Cross Compiler ¶

クロス・コンパイラーは、 Forth カーネルをブート・ストラップするために使用されます。 Gforth は、 外部インタープリターやコンパイラなどの重要な部分を含め、 ほとんどが Forth で書かれているため、 開始するにはコンパイル済みの Forth コードが必要です。 クロス・コンパイラを使用すると、 他のアーキテクチャ用の新しいイメージを作成でき、 別の Forth システムで実行することもできます。
