Appendix A Bugs ¶

既知のバグは、 Gforth ディストリビューションのファイル BUGS に記載されています。

バグを見つけた場合は、 https://savannah.gnu.org/bugs/?func=addbug&group=gforth を通じてバグ・レポートを送信してください。

  • バグを再現するプログラム (または一連のキーボード・コマンド)。
  • バグのある振る舞いを構成すると思われる内容の説明。
  • 使用される Gforth バージョン (対話型 Gforth セッションの開始時に表示されます)。
  • マシンとオペレーティング・システム (Unix システムでは uname -a がこの情報を報告します)。
  • インストール・オプション (構成オプションは config.status の先頭にあります) と構成 (configure 出力または config.cache)。
  • あなた (またはインストーラ) が Gforth ソースに加えた変更 (ある場合) の完全なリスト。

バグ報告の詳細なガイドについては、 How to Report Bugs in GNU C Manual を参照してください。
