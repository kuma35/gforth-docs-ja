14 Image Files ¶

イメージ・ファイルは、 Forth のディクショナリーのイメージ、 つまり、 ディクショナリ内に存在するコンパイルされた Forth コードとデータを含むファイルです。 慣例により、 イメージ・ファイルには拡張子 .fi を使用します。
