2.2 Leaving Gforth

bye または (行の先頭で、) Ctrl-d または (--die-on-signal オプションを指定して Gforth を起動した場合、) Ctrl+C を入力すると、 Gforth を終了できます。 Gforth を終了すると、 あなたの定義とデータはすべて破棄されます。 Gforth を終了する前にシステムの状態を保存する方法については、 Image Files を参照してください。

doc-bye
