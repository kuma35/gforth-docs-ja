Gforth

このマニュアルは、 標準 Forth 言語の高速で移植可能な実装である Gforth (バージョン 0.7.9_20240418, April 18, 2024)用です。 これはリファレンス・マニュアルとして機能しますが、 Forth の概要と Forth チュートリアルも含まれています。
